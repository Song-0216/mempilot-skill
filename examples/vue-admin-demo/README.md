# Vue 管理系统示例

这是一个简化示例，用于展示 MemPilot 如何从代码结构生成中文项目记忆。

## 初始化前

项目中只有代码，没有专门的 AI 项目记忆。示例结构见：

- [`初始化前/项目结构.txt`](初始化前/项目结构.txt)

## MemPilot 的识别结果

MemPilot 没有机械地为 `views/`、`api/`、`stores/` 各建一份记忆，而是识别出三个长期边界：

- 客户管理
- 产品管理
- 认证与权限

## 初始化后

示例生成内容位于 [`初始化后/`](初始化后/)：

```text
初始化后/
├── AGENTS.md
└── 项目记忆/
    ├── 当前状态.md
    ├── 项目索引.md
    ├── 调试记忆.md
    ├── 模块/
    │   ├── 客户管理.md
    │   ├── 产品管理.md
    │   └── 认证与权限.md
    ├── 决策/
    ├── 版本记录/
    └── 归档/
```

这不是固定模板。真实项目的模块名称和内容应由 AI 根据代码、路由、接口、数据模型和用户任务生成。
