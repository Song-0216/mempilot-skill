# 认证与权限

## 模块职责

处理登录状态、Token、角色权限和路由访问控制。

## 当前有效结构

- 登录页面位于 `src/views/login/`
- 状态位于 `src/stores/auth.ts`
- API 位于 `src/api/auth.ts`
- 路由守卫位于 `src/router/index.ts`

## 关键流程

1. 登录后保存 Token
2. 拉取用户角色
3. 路由守卫判断访问权限

## 已确认约束

- 页面不能自行绕过路由权限判断
- 退出登录必须同时清理用户状态

## 依赖关系

- 被客户管理和产品管理依赖

## 当前风险

- Token 失效后的并发请求处理尚未统一

- `src/router/index.ts`
- `src/api/auth.ts`
