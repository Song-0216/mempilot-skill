# 产品管理

## 模块职责

管理产品、分类、规格和产品图片。

## 当前有效结构

- 页面位于 `src/views/products/`
- API 位于 `src/api/products.ts`
- 新增和编辑复用 `ProductForm`

## 关键流程

1. 加载产品与分类数据
2. 将接口数据适配为表单结构
3. 校验并提交产品数据

## 已确认约束

- 不修改现有后端字段名称
