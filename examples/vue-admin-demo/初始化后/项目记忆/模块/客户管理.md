# 客户管理

## 模块职责

管理客户资料、联系人和跟进记录。

## 当前有效结构

- 页面位于 `src/views/customers/`
- API 位于 `src/api/customers.ts`

## 关键流程

1. 列表查询客户
2. 新增或编辑客户资料
3. 保存跟进记录

## 已确认约束

- 客户邮箱允许为空
- 跟进记录只追加，不覆盖历史

## 依赖关系

- 依赖认证与权限模块提供当前用户

## 当前风险

- 跟进记录尚未支持附件

## 相关代码位置

- `src/views/customers/`
- `src/api/customers.ts`
