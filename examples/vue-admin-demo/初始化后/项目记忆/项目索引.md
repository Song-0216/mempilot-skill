# 项目索引

## 项目定位

基于 Vue 3 的内部客户与产品管理系统。

## 核心技术

- Vue 3
- TypeScript
- Vite
- Pinia

## 模块语义路标

| 模块 | 触发关键词 | 文件 |
|---|---|---|
| 客户管理 | 客户、联系人、跟进、线索、CRM | `模块/客户管理.md` |
| 产品管理 | 产品、分类、规格、图片、编辑、上架 | `模块/产品管理.md` |
| 认证与权限 | 登录、Token、角色、权限、路由守卫 | `模块/认证与权限.md` |

## 重要文件

- 当前状态：`当前状态.md`
- 调试记忆：`调试记忆.md`
- 决策：`决策/`
- 版本记录：`版本记录/`
- 归档：`归档/`

## 全局规则

- 接口字段名称以当前后端契约为准
- 时间字段在提交前统一格式化
