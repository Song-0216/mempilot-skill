# Agent 原生入口

入口只做适配，不复制完整 Skill，不自动附加全部记忆。

## Codex / GPT

在 `AGENTS.md` 中合并：先读当前状态；搜索语义路标；只读匹配章节；默认不读版本和归档；结束时三问法局部写回。

## Claude Code

在 `CLAUDE.md` 中加入等价短规则，保留已有 Claude 专属规则。不要自动导入全部模块或完整索引。

## Cursor

创建或更新 `.cursor/rules/项目记忆.mdc`，使用等价短规则，不自动绑定全部记忆文件。

## 切换工具

只新增新工具入口，不重建项目记忆，不删除旧入口。若无法判断当前 Agent，用 `AGENTS.md` 回退并在当前状态标记待确认。
