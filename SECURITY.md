# 安全说明

## 支持范围

MemPilot 是一个纯 Skill，主要通过 Agent 的文件与 Git 能力工作。它不会主动上传数据，也不运行后台服务。

## 敏感信息

MemPilot 的规则禁止将以下信息写入项目记忆：

- API Key、访问 Token 和密码
- 数据库连接串
- 私钥、证书私钥和 PEM 内容
- 用户隐私、客户资料和生产数据
- `.env` 文件正文

但模型仍可能犯错。请在提交代码或公开仓库前审查生成的 `项目记忆/`。

## 报告安全问题

请不要在公开 Issue 中发布真实密钥、私有仓库内容或可利用漏洞细节。

可先创建不含敏感信息的安全报告，说明：

- 问题类型
- 受影响的 Skill 模式
- 可公开的复现步骤
- 可能造成的影响

仓库维护者可在 GitHub 启用 Private Vulnerability Reporting 后使用私密渠道处理。
