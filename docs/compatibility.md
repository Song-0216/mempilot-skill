# 兼容性说明

MemPilot 是纯文本 Skill，兼容性取决于 Coding Agent 是否支持：

- 项目文件读取与写入
- 代码检索和代表性片段读取
- 局部文件编辑或精确章节替换
- Shell / Git 命令（可选但推荐）
- Skill 或项目规则自动发现

## 推荐模型能力

MemPilot 不绑定具体模型名称。推荐使用具备以下能力的模型：

- 能根据路由、数据模型、服务层和测试推断业务边界
- 能区分代码证据、推测和待确认信息
- 能稳定执行工具调用、局部 Patch 和最小范围检索
- 能遵守读取预算、三问写入和破坏性变更确认

## 能力分级与降级行为

| 能力级别 | 可执行范围 | 必须限制 |
|---|---|---|
| 强代码理解与可靠工具调用 | 自动初始化、日常增量、快速维护、定向一致性检查 | 结构变化仍需确认 |
| 中等代码理解 | 初始化后请用户确认模块；可维护已有模块 | 不自动拆合模块或推翻 ADR |
| 轻量模型 | 当前状态更新、已有章节的证据化小 Patch | 不自动规划复杂模块，不填充无证据内容 |
| 无可靠检索或局部编辑 | 只读报告和建议 | 不推荐自动维护项目记忆 |

当模型或工具能力无法判断时，默认按更保守一级执行。低置信时宁可生成 2～5 个高置信模块并列出候选，也不要生成大量空洞模块。

## 设计适配

| Agent / 工具 | 项目入口建议 | 状态 |
|---|---|---|
| OpenAI Codex / GPT 编程 Agent | `AGENTS.md` | 设计支持，等待更多真实项目验证 |
| Claude Code | `CLAUDE.md` | 设计支持，等待更多真实项目验证 |
| Cursor | `.cursor/rules/项目记忆.mdc` | 设计支持，等待更多真实项目验证 |
| 其他 Coding Agent | 原生项目规则或 `AGENTS.md` 回退 | 实验性 |

## 已知差异

- 部分工具不支持按行范围读取，可能退化为读取完整短文件
- 部分工具没有可靠 Patch 能力，应先报告限制，避免重写长文件
- Agent 对 Skill 自动触发的规则不同
- Git 不可用时，维护依据会话文件记录、用户任务和定向检查；文件时间只作辅助
- 轻量语义漂移抽查只能发现有代表性代码证据的变化，不等同于完整静态分析

欢迎通过 Issue 提交模型、Agent、项目类型和实际表现。
